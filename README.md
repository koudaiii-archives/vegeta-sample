docker-vageta
----

## REF

https://github.com/tsenart/vegeta

### How to install

* install docker and docker-compose

* get targert ip

```
 docker-machine ip default
 192.168.99.100
 ```

* edit targets.txt

 ```
 $ vim targets.txt
 ```

```
$ docker-compose build
```

### Use

```
 $ docker-compose up
```

* display help in vegeta
